Connect YouTube

Each user creates their own Google Cloud OAuth client. shortsmith never sees your credentials. Free quota covers 6 uploads/day per project.

  1. Create a Google Cloud project (or pick an existing one). Open Cloud Console ↗
  2. Enable YouTube Data API v3. Click the link, hit "Enable". Enable API ↗
  3. Configure the OAuth consent screen. User type: External. Add yourself as a Test user. Consent screen ↗
  4. Create OAuth client credentials. Click + Create Credentials → OAuth client ID → Desktop app → Create → Download JSON. Credentials ↗
  5. Drop the downloaded JSON here:
    Drop client_secret_*.json or click to pick
  6. Click Connect. Browser opens for sign-in. Allow access. Done.
